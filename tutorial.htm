Tutorial

Brief Introduction

LOGO is a tool that helps beginners to learn how to program. By programming the turtle, user can manipulate the "Turtle" (which is represented as an triangle in this design) on the canvas. The turtle will leave the trace wherever it goes and these traces will be a beautiful pattern.

Get in a dialogue with PiE

First Try

  • Type in FD 150 and Click the Talk. See what happen?
  • Great! The "turtle" just move forward 150 steps! And PiE confirmed your command in the response box.
  • Then Type in LT 90 and Click the Talk. You can also type the Enter on the keyboard instead of the Click.
  • You will then see the "turtle" turn left 90 degrees!
  • Next, let's try move the turtle quicker and more fluently by using repeat.
  • Type in REPEAT 3 [FD 150 LT 90].
  • Yeah! You got a square! right?
  • You can clear the screen by CS

Move the turtle

  • Basically, the turtle can move forward and backward. Try FD 150 and BK 150.
  • Besides, staight move, the turtle can turn clockwise(right) or anti-clockwise(left). Try LT 90 and RT 90. You can name any degree between 0 to 360.
  • If you do not want the pattern any more, just CS!

Attributes can be modified

  • The turtle can draw with its movements. If you want to change how the trace looks like, please go on with this section.
  • First, let's try to make the line bolder. Type in SETPENSIZE PENSIZE + 1 and then try FD 15. How it look?
  • Then, let's make the line lighter. Type in SETPENSIZE PENSIZE - 1 and try again FD 150. See the difference?
  • You can also change the color of the line by SETPENCOLOR 4. You can change to any other colors like: grey 15 purple 13 green 2 yellow 6 black 0 blue 1 red 4.
  • Test with FD 150. Your turtle might hit the wall if you continued three 150 step forward. After it hit the wall, it will appear from the other side.
  • Sometimes, you just want to move the turtle but not leave any traces. In this case, let's suspend the turtle.
  • Suspend the turtle by PENUP and the turtle will be off the canvas. There will be no trace left when the turtle is off the paper. You can try FD 50 to double check.
  • Put down the turtle by PENDOWN
  • Test with FD 100

Basics in Programming

  • In this section, we will tell you some basics in programming.
    Loop
  • Loop is a faster way to get things done if you want to repeat several commands over and over again. Let's see how to use it.
  • First, let's try these commands one by one. FD 150 LT 90 FD 150 LT 90 FD 150 LT 90 FD 150 LT 90 What do you get? A square! But how to use less commands to realize the same thing?
  • Try these commands one by one
    FD 150 LT 90 REPEAT 3 [FD 150 LT 90]
  • You see. We use 3 commands to draw a square instead of 8 commands.
    Function
  • You can read this part after the next section "Write in Scripts".
  • Function helps to reuse some capsulated commands. Try the following commands:
  TO Square 
    REPEAT 4 [ FD 150 LT 90]
  END
  CS
  Square
  • To experience the benefit from Function, you can go on with
    •   RT 60
  Square

    Write in Scripts

    • You can write down a few sentences to manipulate the turtle at a time instead of being involved in the tedious dialogue. Take the square as an example. Try copy-paste the following commands in the Multi-line Mode by clicking on the arrow beside the Talk button.
    SETPENCOLOR 4
SETPENSIZE PENSIZE + 1
REPEAT 4 [ FD 150 LT 90]
    • After the copy-paste, click on the Talk button. What do you get?
    • You can revise the script in the Multi-line mode by adding or deleting in the box. Try add REPEAT 36 [ after SETPENSIZE PENSIZE + 1 and LT 10 and ] after the original ]. What do you get?
    SETPENCOLOR 4
SETPENSIZE PENSIZE + 1
REPEAT 36 [ REPEAT 4 [ FD 150 LT 90]]
    • What? Did not see it clearly. Let's try it step by step. In the third command, change "36" into 1, 2, 4, 8, 12, 16 respectively. Do you understand now?
